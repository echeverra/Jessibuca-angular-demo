# Jessibuca-angular-demo

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 16.1.3.

## Environment Prepare

Install node_modules:

```
npm install
```

## Start project

```
ng serve
```

Open http://localhost:4200 to view it in the browser.
